Smart-Review 复习管理器 (算法模块)
=================================

SmartReview的算法模块，所有与算法相关的内容都在此模块内实现。

规划内容
-----------

- 根据单词之间的相似度自动计算相似单词
- 根据用户反馈自动设置复习间隔
- 采用倒排索引实现的搜索功能

已实现内容
------------

- 使用NLTK库还原句子中的单词，并获取单词的分类标签
